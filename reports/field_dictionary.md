# WoS 字段字典

| 字段名 | 原始标签 | 说明 |
|--------|----------|------|
| Doc_Type | PT | 文献类型 |
| Authors | AU | 作者列表（分号分隔） |
| Title | TI | 文献标题 |
| Journal | SO | 来源期刊 |
| Author_Keywords | DE | 作者提供的关键词 |
| Keywords_Plus | ID | WoS自动生成的关键词 |
| Abstract | AB | 摘要内容 |
| Author_Address | C1 | 作者机构地址 |
| References | CR | 参考文献列表 |
| Year | PY | 出版年份 |
| Times_Cited | TC | 总被引频次 |
| DOI | DI | 数字对象标识符 |
| Volume | VL | 卷号 |
| Issue | IS | 期号 |
| Page_Start | BP | 起始页码 |
